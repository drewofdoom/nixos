# drewofdoom's NixOS config

This is my personal NixOS configuration. It was modified from the [nnn-starter](https://github.com/floatdrop/nnn-starter) by floatdrop. Please only use my repo as a reference and do not copy it directly. You should start from nnn-starter and modify it to suit your needs.

## What's changed

A lot has been modified in this repo. For one, I'm targeting the latest noctalia and niri versions. Additionally, I'm set up here for multiple machines, and they are hardcoded into my flake.nix.

I've added flatpak support, set fish as the default shell, changed the theming, and added pro-audio stuff. You likely don't need all of this stuff.
